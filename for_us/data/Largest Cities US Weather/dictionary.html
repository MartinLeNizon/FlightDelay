Weather Data Dictionary

Column Description
city The name of the city where the weather data was recorded.
date The date when the weather data was recorded (in DD.MM.YYYY format).
tavg Average temperature for the day, measured in degrees Fahrenheit.
tmin Minimum temperature for the day, measured in degrees Fahrenheit.
tmax Maximum temperature for the day, measured in degrees Fahrenheit.
prcp Precipitation level for the day, measured in millimeters.
snow Snowfall amount for the day, measured in millimeters.
wdir Average wind direction, measured in degrees (0° to 360°).
wspd Average wind speed for the day, measured in kilometers per hour.
wpgst Maximum wind gust speed for the day, measured in kilometers per hour. If not available, marked as "NULL".
pres Average atmospheric pressure for the day, measured in hPa (hectopascals).
tsun Total sunshine duration for the day, measured in minutes. If not available, marked as "NULL".
